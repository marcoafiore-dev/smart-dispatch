## Deploy en Vercel (30 segundos)

### Opción A — Arrastrá la carpeta a Netlify
1. Ve a https://netlify.com
2. Arrastrá la carpeta `public/` directo a la página
3. URL lista al instante

### Opción B — GitHub + Vercel
1. Subí este repo a GitHub
2. En Vercel: New Project → Import → Deploy
3. Auto-redeploy en cada push

### Opción C — Vercel CLI
```bash
npm i -g vercel
vercel --prod
```
